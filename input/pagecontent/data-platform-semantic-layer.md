La couche sémantique décrit les données répondant aux besoins métiers préalablement identifiés de sorte que elles soient le plus réutilisable possible afin d'améliorer leur valeur.

Le modèle de donnée du Hub d'intégration a été dérivé des ressources FHIR servant de définition de base aux profils conçu dans le cadre du [processus de standardisation des données au format FHIR](https://aphp.github.io/IG-FHIR-EDSH-SOCLE-COMMUN/index.html).
